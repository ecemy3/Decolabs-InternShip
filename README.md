# Decolabs Internship

This repository contains data analytics projects completed during my internship at DecodeLabs. The projects demonstrate end-to-end data analysis skills including data cleaning, exploratory analysis, SQL querying, and data visualization.

## Projects

- `projects/project1-data-cleaning`: Data cleaning pipeline including handling missing values, removing duplicates, and standardizing formats.
- `projects/project2-eda-sales-analysis`: Exploratory Data Analysis with statistical summaries, trend analysis, and business insights supported by visualizations.
- `projects/Project3`: SQL-based data analysis using SELECT, WHERE, GROUP BY, and aggregation functions to extract actionable insights.
- `projects/project4-visual-storytelling-sales`: Insight-to-story visual analytics project with action-title charts and SCR-based business reporting.

## Skills Demonstrated

- Data Cleaning & Preparation
- Exploratory Data Analysis (EDA)
- SQL Data Analysis
- Data Visualization & Storytelling
