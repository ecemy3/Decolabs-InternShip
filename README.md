# Decolabs Internship

This repository contains data analytics projects completed during my internship at DecodeLabs. The projects demonstrate end-to-end data analysis skills including data cleaning, exploratory analysis, SQL querying, and data visualization.

## Projects

- `project1-data-cleaning`  
  Data cleaning pipeline including handling missing values, removing duplicates, and standardizing formats.

- `project2-eda-sales-analysis`  
  Exploratory Data Analysis with statistical summaries, trend analysis, and business insights supported by visualizations.

- `project3-sql-analysis`  
  SQL-based data analysis using SELECT, WHERE, GROUP BY, and aggregation functions to extract actionable insights.

- `project4-data-visualization`  
  Data storytelling through visualizations, highlighting key insights using appropriate charts.

## Skills Demonstrated

- Data Cleaning & Preparation

- SQL Data Analysis
